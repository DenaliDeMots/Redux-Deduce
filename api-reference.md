## Primitive Actions

### SET 

Replace the root value in the state-tree.

Works on: Numbers, Booleans, Strings

```javascript
D.SET({value: 5}); // Sets root state to 5
```

### SET\_IN 

```javascript
D.SET_IN({value: 5, key}); // Sets root state to 5
```
